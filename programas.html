Exemplo de Programa em R

A seguir, apresentamos um exemplo de programa em R que calcula o fatorial de um número e uma aproximação do número de Euler usando a série de Taylor.

Programa: Cálculo de Fatorial e Número de Euler

# Função para calcular o fatorial
fatorial <- function(n) {
  if (n == 0) {
    return(1)
  } else {
    return(n * fatorial(n - 1))
  }
}

# Função para calcular o número de Euler usando a série de Taylor
calcular_euler <- function(n) {
  euler <- 1
  for (i in 1:n) {
    euler <- euler + 1 / fatorial(i)
  }
  return(euler)
}

# Testando as funções
numero <- 5
cat("O fatorial de", numero, "é", fatorial(numero), "\n")
cat("A aproximação do número de Euler usando", numero, "termos é", calcular_euler(numero), "\n")
